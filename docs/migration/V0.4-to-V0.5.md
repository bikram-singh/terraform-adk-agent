# Migration: v0.4 to v0.5

Version 0.5 introduces a plugin-based Multi-Service Generator Framework.

The public ADK tool remains `generate_gcs_terraform_project`, so existing
Version 0.4 prompts continue to work.

No files need to be deleted for this overlay. The Version 0.5 engine uses the
new `terraform_agent/generators/gcs/` plugin package.
