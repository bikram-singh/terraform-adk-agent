# Migration: v0.5 to v0.6

Version 0.6 adds a production-ready Cloud Run plugin while keeping the
existing GCS workflow compatible.

- Cloud Run plugin metadata, templates, generator, ADK tool, and tests

## Updated
- Generator bootstrap

## Security defaults
- Authentication required
- Dedicated runtime service account
- No service account keys
- Deletion protection enabled
- Secret values excluded from Terraform
- Secret-level IAM
- Public access only by explicit opt-in
