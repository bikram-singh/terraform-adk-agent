# Migration: v0.7.2 to v0.8

Version 0.8 adds the enterprise Google Kubernetes Engine generator plugin.

## Added

- Standard and Autopilot GKE generation
- Private nodes and private endpoint default
- Workload Identity Federation for GKE
- Gateway API channel
- Managed Service for Prometheus
- Standard node-pool autoscaling and hardening
- Dedicated Standard node service account
- Artifact Registry support
- GKE ADK tool
- GKE unit tests

## Compatibility

Existing GCS, Cloud Run, MCP, and structured Registry functionality remains
unchanged.

## Existing infrastructure requirements

The generated project expects an existing VPC, subnet, Pod secondary range,
and Service secondary range.
