# Migration: v0.6.1 to v0.7

Version 0.7 integrates HashiCorp's official Terraform MCP Server with Google
ADK through Docker and stdio.

The integration enables only the public `registry` toolset and explicitly sets
`ENABLE_TF_OPERATIONS=false`. No HCP Terraform token is required.

Required environment variables:

    TERRAFORM_MCP_ENABLED=true
    TERRAFORM_MCP_DOCKER_IMAGE=hashicorp/terraform-mcp-server:1.0.0
    TERRAFORM_MCP_TIMEOUT_SECONDS=60
