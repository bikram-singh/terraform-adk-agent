# Migration: v0.9 to v0.9.1

Adds the private Cloud SQL generator.

Supported:

- PostgreSQL and MySQL
- Private IP only
- Regional or zonal availability
- Automated backups
- PostgreSQL PITR
- Query Insights
- IAM database authentication
- Optional CMEK
- Maintenance windows
- Deletion protection
- Application database creation

The VPC and Private Service Access connection must already exist.
